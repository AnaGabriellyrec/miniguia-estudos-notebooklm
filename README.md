# miniguia-estudos-notebooklm

Projeto prático para aliar pensamento crítico, curadoria de fontes e organização do conhecimento para criar um Caderno Temático no NotebookLM.

//Objetivo:

*Criação de um caderno temático no NotebookLM.*

A partir desse material, são definidos objetivos de estudo, elaboradas perguntas estratégicas e testadas variações de prompts, registrando as respostas e suas referências.

//Tema:

Introdução ao Python para iniciantes

*Criação de perguntas simples (nível iniciante)

*Comparação entre respostas geradas

//Dificuldades encontradas:

*Necessidade de refinar os prompts para obter respostas mais completas
*Diferença de qualidade dependendo da forma da pergunta
